## About Me

I'm a Mechatronics Engineering graduate working as a **Control & Automation Engineer** at ARIL Technologies, Lagos, where I design, programme, and document industrial control systems for the oil and gas sector.

My work spans the full lifecycle of a control system — from **Functional Design Specifications** and **P&ID interpretation** through **PLC programming**, **SCADA configuration**, **IO list validation**, and **commissioning documentation**. I have hands-on experience with Siemens and Mitsubishi platforms, and I'm actively building expertise.

## Featured Projects

### 🏭 [GDU Pilot Plant Control System Simulation](./gdu-pilot-plant-simulation)
> Gas Dehydration Unit control system built entirely in simulation — from FDS through PLC programme to HMI

A full-scope control system engineering project for a Gas Dehydration Unit (GDU) pilot plant. Covers dual-vendor PLC architecture (Mitsubishi Q06UDVCPU master + Schneider M340 slaves), RS485 Modbus RTU communications, molecular sieve regeneration sequence design, 204-tag IO register workbook, and a complete Structured Text state machine implementation.
